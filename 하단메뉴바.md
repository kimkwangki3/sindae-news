# 웹사이트 하단 메뉴바 (Bottom Navigation Bar)

깔끔한 선형 아이콘 + 따뜻한 뉴트럴 톤으로 디자인한 하단 메뉴바입니다.
글자는 원본 그대로(홈 · 기사 · 상권 · 지역단체 · 나눔마켓 · 게시판) 유지했습니다.

아래 코드를 VSCode에서 그대로 복사해 사용하세요.

---

## 2. 사용 팁

- **활성 탭 변경**: `is-active` 클래스를 원하는 `<a>` 로 옮기면 됩니다. (인디케이터 `<span class="bottom-nav__indicator">` 도 같은 항목 안에 두세요.)
- **포인트 컬러 변경**: CSS 의 `#c0563f` 값을 원하는 색으로 바꾸면 활성 색/인디케이터가 함께 바뀝니다.
- **링크 연결**: 각 `<a href="#">` 의 `#` 을 실제 페이지 주소로 바꾸세요.
- **하단 고정**: 항상 화면 맨 아래 붙이고 싶다면 `.bottom-nav` 에 아래를 추가하세요.
  ```css
  position: fixed;
  bottom: 0;
  left: 50%;
  transform: translateX(-50%);
  z-index: 100;
  ```
